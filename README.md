# party_llama_dos
A static site for info about our wedding! yay!

This site is built with [Middleman](https://middlemanapp.com/)

## To get it running locally: 
1. Clone the repo
1. `gem install middleman`
1. `bundle install`
1. `middleman server`

## Deployment
1. run `middleman build`
1. Commit the changed files
1. push to master!
